# psiKevelin Landing Page

Repositório para a landing page da psicóloga Kevelin Camilo. Este site apresenta informações sobre a profissional e seus serviços de psicologia, proporcionando uma experiência online informativa e agradável para os visitantes.

## Visite o Site
[psikevelincamilo.com.br](https://psikevelincamilo.com.br)

## Funcionalidades
- **Informações Profissionais**: Apresenta as qualificações e especialidades da psicóloga.
- **Contato**: Links de redirecionamento para redes sociais, whatsapp business e email.
- **Responsividade**: Layout adaptável para desktop e dispositivos móveis.

## Tecnologias Utilizadas
- **HTML5**: Estrutura do conteúdo.
- **CSS3**: Estilização visual e responsividade.
- **Bootstrap**: Framework para design responsivo e componentes estilizados.

## Estrutura do Projeto
- `index.html`: Página principal.
- `css/`: Arquivos de estilo.
- `img/`: Imagens usadas no site.
